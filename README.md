# 👑 Glam Anime Dress-Up Game
A fun AI powered anime dress-up game with a beauty contest 
judging system built with Gemini API and Gradio.

Go through 3 stages:
- 🌸 Skincare routine
- 💄 Makeup look
- 👗 Choose your outfit
- 🏆 Enter the beauty contest and get AI judged!

## 🛠️ Built With
- Google Gemini API
- Gradio
- Matplotlib
- Google Colab

## 🚀 How to Run
1. Clone this repo
2. pip install gradio google-generativeai matplotlib pillow
3. Add your Gemini API key
4. Run app.py

## 📝 Note
Character visuals are matplotlib drawings — 
future version will use real anime sprite art!
